# Database (데이터베이스)
Database(DB)는 여러 사람이 공유하여 하용할 목적으로 체계화해 통합, 관리하는 데이터의 집합이다.

---
- [DBMS](DBMS.md)와는 엄연히 다르다

## Database의 장점
- 데이터 중복 최소화
- 데이터 공유
- 데이터 저장 공간 절약
## Database의 단점
- 시스템의 복잡함
- 대용량 디스크로 엑세스가 집중되면 과부하 발생
- 데이터베이스 전문가 필요
