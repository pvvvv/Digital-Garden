# Hibernate (하이버네이트)
Hibernate ORM은 자바 언어를 위한 객체 관계 매핑 프레임워크이다

---
- *JPA의 구현체로, JPA 인터페이스를 구현하며, 내부적으로 *JDBC API를 사용.

## 장단점
- [ORM](ORM.md)의 장단점

## *JPA 
- 자바 애플리케이션에서 관계형 데이터베이스를 사용하는 방식을 정의한 인터페이스

## *JDBC
- 자바 프로그래밍 언어와 다양한 데이터베이스 SQL 또는 테이블 형태의 데이터 사이에 독립적인 연결을 지원하는 표준  
-> DB 작업을 위한 표준
