# EJS
EJS는 단순히 *Embedded JavaScript 를 의미.  
node.js 진영에서 많이 사용하는 뷰 템플릿엔진이다.
##### *Embedded : 간직하다, 끼워넣다

---
- 동적 콘텐츠를 생성하거나 실시간 업데이트와 관련된 것을 제공하는 경우 코드로드를 크게 줄일 수 있다.


> 주석 : **<%# ... %>**  
> JS 코드 : **<% ... %>**  
> 변수 출력(html escape 처리: >를 $gt로 변환) : **<%= ... %>**  
> 태그내부 공백 제거 : **<%_ ... _%>**  
> html escape안하고 변수 출력 : **<%- ... %>**
